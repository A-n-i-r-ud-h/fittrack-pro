💪 FitTrack Pro
A full-stack fitness tracking web application where users can log workouts, track progress, and view analytics.

🚀 Features
User Authentication (JWT आधारित login/register)
Add / Delete Workouts
Dashboard with summary
Weekly stats visualization
Secure API with Spring Security
🛠 Tech Stack
Frontend:

React (Vite)
Tailwind CSS
Axios
Backend:

Spring Boot
Spring Security (JWT)
Hibernate / JPA
MySQL
🔐 Authentication
JWT-based authentication
Secure API routes
Token stored in localStorage
📦 API Endpoints
/api/auth/register
/api/auth/login
/api/workouts
/api/workouts/summary
/api/workouts/weekly
⚙️ Setup Instructions
Backend
cd backend
mvn spring-boot:run
